# Source directory

This directory contains source code for the Katan project.

The source code is divided into the following components:

- apps (Applications source code)
  - kt_app (Katana Application source code)
  - os (Operating System specific function code to mbe implemented by the Appp)
- hub (HUB library source code)
- interface (Header files interfacing the various components)
- KtLib (Katana library source code)
- scripts (Scripts for running various commands in an automated fashion - current version limited to Linux)
