# Katana Visualizer

This directory contains code, resources, and assets for the Katana Visualizer, a Python Flask-based web application for demonstrating Katana in action.

The Python code interacts with Kata (Kt App and Ktlib) through IPC (Unix domain sockets) - hence, once needs to ensure the kt_|app_ipc.elf is run before the Visualizer.

Please refer to the README.md in the <PROJECT_ROOT>/src/scripts/ directory for details on how to run the Katana Visualizer.
